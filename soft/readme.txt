The directory should contain following software in order to have the utility properly work:
 - '7-Zip' folder with the ready to use tool
 - 'PythonAddons' with file 'get-pip' inside that needs for python to download required packages
 - 'openjdk-11.0.2_windows-x64_bin.zip' the JDK installer
 - 'postgresql-12.0-1-windows-x64-binaries.zip' the PostgreSQL database installer
 - 'python-3.7.4.zip' the Python installer
